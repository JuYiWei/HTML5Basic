1.1 浏览器支持

  • 1、HTML 5 浏览器支持
  • 2、添加新元素
  • 3、IE 浏览器

1、HTML 5 浏览器支持

  • 现在的浏览器几乎都支持 HTML 5；
  • 浏览器对【未知】的元素视为【内联】元素处理；
  • HTML 5 新增的 8 个语义元素为【块级】元素，为了兼容旧版，可手动设置 CSS；
header, section, footer, aside, nav, main, article, figure {
    display: block; 
}

2、添加新元素

<head>
    <style>
        myTag {
            display: block;
            color: red;
        }
    </style>
</head>
<body>

<myTag>
    我的元素
</myTag>
